Производство
  • Завершение смены - предоставляет права на завершение смены.
  • Мастер смены - предоставляет права на ввод данных оперативного учета производства.
  • Просмотр данных оперативного учета производства - предоставляет права на просмотр отчетов по данным оперативного учета производства.
  • Сертификация – предоставляет права для работы c подсистемой сертификации.
  • Технолог – предоставляет права на ввод данных нормативной системы производства.
  • Экономист цеха – предоставляет права на отражение операций управления производством в управленческом учете.

Торговля и склад
  • Ввод корректировок поступления – предоставляет права на оформление документа "Корректировка поступления". Назначается совместно с ролью "Менеджер по закупкам".
  • Ввод корректировок реализации – предоставляет права на оформление документа "Корректировка реализации". Назначается совместно с ролью "Менеджер по продажам".
  • Кладовщик – предоставляет права на оформление складских операций по управленческому учету.
  • Менеджер по заказам - предоставляет права на работу с подсистемой заказов. Может использоваться как самостоятельно, так и совместно с другими ролями.
  • Менеджер по закупкам – предоставляет права для работы с подсистемой управления закупками.
  • Менеджер по продажам – предоставляет права для работы с подсистемой управления продажами.
  • Розничные продажи – предоставляет права на оформление документов по розничным продажам. Назначается совместно с ролью "Менеджер по продажам".
  • Ценообразование – предоставляет права для работы с подсистемой ценообразования.
Финансы
  • Банковские операции - предоставляет права на выполнение операций с банковскими документами.
  • Ввод заявок на расходование средств - предоставляет права на оформление документов "Заявка на расходование средств" и "Закрытие заявок на расходование средств".
  • Выплата зарплаты - назначается совместно с ролями "Кассир", "Банковские операции". Предоставляет права на оформление документов:
    • "Приходный кассовый ордер" с видами операций:
      • Возврат денежных средств работником
      • Расчеты по кредитам и займам с работниками
    • "Расходный кассовый ордер" с видами операций:
      • Расчеты по кредитам и займам с работниками
      • Выплата заработной платы по ведомостям
      • Выплата заработной платы работнику
      • Выплата депонированной заработной платы
      • Выдача денежных средств в кассу ККМ
    • "Платежное поручение исходящее" с видами операций:
      • Расчеты по кредитам и займам с работниками
      • Перечисление заработной платы
  • Денежные документы - предоставляет права на выполнение операций по поступлению и выдаче денежных документов. Назначается совместно с ролью "Кассир".
  • Кассир - предоставляет права на оформление кассовых документов.
  • Обмен данными с программами Клиент-банк - назначается пользователю, если ему необходимо иметь возможность использовать обработку "Клиент-банк".
  • Подотчетные лица - предоставляет права на оформление документов по подотчетным лицам. Должна использоваться совместно с другими ролями "Кассир", "Банковские операции", "Менеджер по закупкам".
  • Согласование заявок на расходование ДС - предоставляет права на согласование заявок.
  • Финансист – предоставляет права на отражение операций управления денежными средствами в управленческом учете.  

Зарплата и управление персоналом
  • Кадровик регламентированных данных – предоставляет права на работу с кадровыми документами по организациям, персонифицированным и воинским учетом работников организаций, штатным расписанием организаций.
  • Кадровик управленческих данных – предоставляет права на работу с кадровыми документами по предприятию, документами по обучению, планированию и утверждению отпусков, анкетированию.
  • Менеджер по набору персонала – предоставляет права на работу с кандидатами, с подсистемой анкетирования, с кадровым планом предприятия.
  • Расчетчик регламентированной зарплаты – предоставляет права на расчет регламентированных начислений, расчет налогов, отражение зарплаты в бухгалтерском учете.
  • Расчетчик управленческой зарплаты – предоставляет права на расчет управленческих начислений.

Внеоборотные активы и спецодежда
  • Учет ОС и НМА – предоставляет права на отражение операций с ОС (в т.ч. управление обслуживанием оборудования) и НМА в управленческом учете.
  • Учет спецодежды - предоставляет права к объектам подсистемы "Спецодежда и спецоснастка".

Анализ и планирование
  • Бюджетирование – предоставляет права для работы с подсистемой бюджетирования.
  • Планирование – предоставляет права для работы с подсистемой планирования.
  • Учет затрат - предоставляет права к данным учета затрат и себестоимости номенклатуры по управленческому и регламентированному учетам.

Регламентированный учет
  • Бухгалтер МСФО – предоставляет права на отражение хозяйственных операций в МСФО.
  • Выполнение регламентных операций по регламентированному учету - предоставляет права для проведения регламентных документов по регламентированному учету (БУ и НУ).
  • Контролируемые сделки - предоставляет права для подготовки уведомлений по контролируемым сделкам.
  • Отражение в регламентированном учете - предоставляет права для проведения документов по бухгалтерскому и налоговому учету.
  • Право на защищенный документооборот с контролирующими органами (ФНС, ПФР) - предоставляет право на использование встроенного механизма информационного взаимодействия с контролирующими органами (ФНС и ПФР) по каналам связи. Роль назначается совместно с одной из ролей: "Регламентированная отчетность", "Кадровик регламентированных данных" или "Расчетчик регламентированной зарплаты".
  • Просмотр данных регламентированного учета - предоставляет права для просмотра данных бухгалтерского и налогового учета, а также права для формирования отчетов подсистемы бухгалтерского и налогового учета.
  • Регламентированная отчетность - предоставляет права для использования регламентированной отчетности. Роль используется совместно с ролью "Отражение в регламентированном учете".
  • Учет НДС - предоставляет права к объектам подсистемы НДС, проведению регламентных операций по НДС и формированию отчетов по НДС.
  • Учет РБП - предоставляет права к учету расходов будущих периодов. 

Настройка НСИ
  • Настройка НСИ - предоставляет права к общим объектам, например, склады, подразделения, учетную политику (упр. учет).
  • Настройка НСИ затрат - предоставляет права к данным подсистемы "Управление затратами".
  • Настройка НСИ (регл) - предоставляет права к данным, относящимся к регламентированному учету.

Роли "Настройка НСИ ..." имеют следующую особенность:

  • Не имеют ограничений на уровне записей
    • к объектам НСИ
    • к объектам, связанным с объектами НСИ, т.е. таким, на которые есть ссылки в объектах НСИ

Роли следует назначать компетентным пользователям.

Торговое оборудование
  • Администратор ККМ - предоставляет права на настройку касс ККМ и закрытие кассовой смены.
  • Использование торгового оборудования - назначается пользователю, если ему необходимо иметь возможность использовать торговое оборудование.
  • Настройка торгового оборудования - назначается пользователю, если ему необходимо иметь возможность настраивать торговое оборудование.
  • Оператор ККМ - предоставляет права на оформление документа по розничным продажам "Чек ККМ".

Административные
  • Администратор пользователей - предоставляет права для администрирования пользователей.
  • Полные права - предоставляет доступ ко всем объектам конфигурации и предназначена для выполнения сервисных операций:
    • обновление ИБ
    • удаление помеченных на удаление объектов
    • работа с журналом регистрации
    • и т.п.
    Не рекомендуется работать в системе на постоянной основе под ролью "Полные права", в том числе выполнять бизнес-операции.
  • Право администрирования - назначается пользователю, если ему необходимо выполнять административные функции и работать монопольно.
  • Установка даты запрета изменения данных - назначается пользователю, если ему необходимо иметь возможность устанавливать дату запрета редактирования данных.

Сервисные
  • Администрирование дополнительных форм и обработок - назначается пользователю, если ему необходимо редактировать справочник "Внешние обработки".
  • Администрирование сохраненных настроек - назначается пользователю, если ему необходимо управлять сохраненными настройками отчетов построенных на базе универсального.
  • Изменение макетов печатных форм - назначается пользователю, если ему необходимо редактировать макеты печатных форм.
  • Использование обмена электронными документами - предоставляет права на использование обмена электронными документами. Назначается совместно с ролями "Менеджер по закупкам", "Менеджер по продажам".
  • Право внешнего подключения - назначается пользователю, если ему необходимо работать с информационной базой через внешнее соединение.
  • Право вывода информации - назначается пользователю, если ему необходимо иметь возможность выводить на печать печатные формы, копировать сформированные печатные формы в буфер обмена или сохранять их виде внешних файлов.
  • Право запуска внешних отчетов и обработок - назначается пользователю, если ему необходимо иметь возможность открывать внешние отчеты и обработки.
    Роль следует назначать только компетентным сотрудникам:
    Использование внешних отчетов или обработок может представлять угрозу безопасности данных системы в виде: разрушения (злонамеренного или из-за некомпетентности), несанкционированного доступа.
  • Право использования электронной почты - назначается пользователю, если ему необходимо работать со встроенной электронной почтой.
  • Просмотр движений документов - назначается пользователю, если ему необходимо смотреть движения документов (в управляемом приложении).
  • Просмотр структуры подчиненности - назначается пользователю, если ему необходимо смотреть структуру подчиненности документов (в управляемом приложении).
  • Режим "Все функции" - назначается пользователю, если ему необходим доступ к меню "Все функции".

Также в конфигурации определена роль "Пользователь". Эта роль дает доступ на вход в программу и назначается всем пользователям автоматически (при изменении ролей пользователей в режиме предприятия).
