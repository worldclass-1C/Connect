Форма фоновое задание предназначена для просмотра всех свойств регламентного задния.

Обычно, требуется только при возникновении проблем в работе фоновых заданий. Возможно провести поверхностный анализ периода выполения для примерной оценки временных затрат и выяснения моментов выполнения.

1. Группа свойств "Основное".

Наименование свойства Назначение свойства
Наименование Наименование регламентного задания. Обычно пустое, и в качестве него применяется синоним, указанный в шаблоне регламентного задания, определенного в конфигураторе. Если синоним шаблона не задан, используется его имя.
Использование

Флажок позволять включить или отключить обработку задания. Полезно, когда нужно ненадолго отключить задание, а расписание менять нет смысла.

При ошибке повторять, раз Число попыток выполнить регламентное задание, если при выполнении происходит ошибка.
При ошибке повторять через, сек Пауза (в секундах) между попытками выполнить регламентное задание, если при выполнении происходит ошибка.

 

2. Группа свойств "Дополнительно".

Наименование свойства Назначение свойства
Идентификатор

Нужен только для продвинутого администратора, знающего внутренний язык 1С. Требуется для исследования нестандартных проблем в работе фоновых заданий, если таковые есть.

Ключ

Значение свойства Ключ, указанного в регламентном задании (на момент создания фонового задания) или другое, указанное явно, если фоновое задание создано программно, а не планировщиком сервера.

Имя метода Имя процедуры в конфигруции, которая выполнятся или выполнялась в фоновом задании. Обычно соответствует свойству "имя метода" соответствующего регламентного задания.

 

3. Группа свойств "Регламентное задание".

Наименование свойства Назначение свойства
Наименование Наименование, если пустое, тогда Синоним, если пустой, тогда Имя регламентного задания, соответствующий фоновому заданию.
Идентификатор Идентификатор регламентного задания, соответствующий фоновому заданию.

 

4. Сообщения пользователю и описание информации об ошибке.

Поле содержит текс сообщений и текст сведений об ошибке, если сообщения и/или ошибки произошли при выполнении фонового задания.
