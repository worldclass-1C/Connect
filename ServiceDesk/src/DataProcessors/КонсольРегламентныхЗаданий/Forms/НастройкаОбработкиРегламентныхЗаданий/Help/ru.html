Форма "Настройка обработки регламентных заданий" предназначена для файловой информационной базы.

Наименование свойства А Назначение свойства
Блокировка регламентных заданий + Установка флажка блокирует обработку регламентных заданий в сеансе, который их обрабатывает, сеанс продолжает работу. Снятие флажка возобновляет обработку.
Автоматически открывать отдельный сеанс обработки регламентных заданий Установка флажка настраивает режим открытия отдельного сеанса обработки регламентных заданий при запуске приложения, если приложение запущено с ключом выполнения регламентных заданий, тогда запуск будет пропущен, т.к. в качестве сеанса, обрабатывающего регламентные задания будет назначен текущий сеанс.
Имя пользователя + Имя того пользователя информационной базы, который будет применен при автоматическом открытии отдельного сеанса обработки регламентных заданий. Если применяется "сквозная" Windows-аутентификация или список пользователей информационной базы пустой, то имя пользователя указывать не требуется.
Пароль пользователя + Пароль пользователя указывается, если указано имя пользователя.
Уведомлять о некорректном состоянии обработки регламентных заданий Уведомлять пользователя, что регламентные задания не выполняются, если установлен флажок "Автоматически открывать отдельный сеанс обработки регламентных заданий".
Период уведомления, минут Период в минутах, через который будет производится проверка состояния обработки регламентных заданий с последующим уведомлением пользователя, если состояние некорректно.

Прим.: колонка "А" содержит знак "+" там, где для изменения свойств пользователь должен иметь право Администрирования.

Команда Описание
Открыть сеанс. Только для файловой ИБ. Открывает отдельный сеанс для обработки регламентных заданий. Фактически, производится запуск отдельного сеанса с параметрами, такими, что сеанс открывает я только для обработки регламентных заданий, считает текущий сеанс родительским, и при его завершении автоматически закрывается.

 

Параметры инфорационной базы при запуске 1С Предприятия 8 для включения автоматической обработки регламентных заданий.

Параметр

 Описание
DoScheduledJobs Обрабатывать регламентные задания в открываемом сеансе.
SkipMessageBox Не показывать предупреждения, в случах, когда сеанс, обрабатывающий задания, уже открыт или есть другие причины, препядствующие обработке заданий в открываемом сеансе.
AloneIBSession Открыть сеанс только для обработки регламентных заданий. Если сеанс для целей обработки уже открыт, то открываемый сеанс будет завершен с предупреждением (или без предупреждения см. SkipMessageBox).

Примеры строки программы в ярылке Windows.

Назначение

 Строка
Включить обработку заданий в открываемом сеансе. "C:\Program Files\1cv82\Bin\1cv8c.exe" /CDoScheduledJobs

Запустить сеанс только для обработки заданий. Например, если каталог информационной базы расположен на отдельном компьютере, то эту строку можно записать в планировщик Windows. Лишний запуск сеанса будет автоматически завершен.

 "C:\Program Files\1cv82\Bin\1cv8c.exe" ENTERPRISE /IBConnectionString"File=""C:\Инфорационная база"";" /NВасилий /PПароль /C"DoScheduledJobs SkipMessageBox AloneIBSession".
